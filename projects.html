Joliet Homepage Remake

I wanted to try on a project that would test my skill with vanilla js and css, no frameworks or libraries allowed! I decided to remake my home town's website without the help of bootstrap. This project gave me a new found love for bootstrap and how it abstracts away all the nitty gritty. Also, flexbox is now my bestfriend. I ultimately had to use jQuery to make the mobile menu but that was something I adapted from the iTube project. It's only the homepage and lacks polish, but it was a lot of work and I'm pretty happy with how it turned out. I only tested it in chrome so your mileage will vary on other browsers.
